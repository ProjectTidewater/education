# Project Tidewater Education

Disclaimer: This is a (very) rough draft and is subject to change.

PTE will issue its own coin. The quantity will be the same as Skycoin as will the price match Skycoins price. Probably will use Oracles to set the price. This will be the initial method for funding. Payment will be in Skycoin and the Skycoin used in the purchase will be only used for the educational tasks that the buyer of the PTE coins wants. If the Skycoins haven't been earned or at least claimed by a student in a certain time the buyer of the PTE has the option to buy back their Skycoin in exchange for their PTE coins.
